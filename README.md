# 初始化项目



```
yarn create react-app react-antd-admin-ts --template=typescript

```

## 添加Antd

```
yarn add antd
```

## 添加 axios

```
yarn add axios

```

## 添加

```
yarn add nprogress @types/nprogress

```

## 添加路由

```
yarn add react-router-dom @types/react-router-dom

```
